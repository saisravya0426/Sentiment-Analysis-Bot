# Sentiment Analysis Chatbot using NLP

## Overview

This project is a **console-based chatbot** that performs **real-time sentiment analysis** on user inputs. It uses Natural Language Processing (NLP) techniques to classify text as positive, negative, or neutral and responds accordingly.

The project demonstrates the application of **OOP principles, NLP, and efficient input handling** to build an interactive system.

---

## Key Features

* Real-time sentiment analysis of user input
* Classification into positive, negative, or neutral sentiments
* Interactive chatbot with dynamic responses
* Built using Object-Oriented Programming (OOP) principles
* Optimized to handle 100+ inputs per minute efficiently

---

## System Workflow

1. **User Input**

   * User enters a message via console

2. **Text Processing**

   * Input is cleaned and processed

3. **Sentiment Analysis**

   * Sentiment is computed using TextBlob

4. **Response Generation**

   * Bot responds based on detected sentiment

---

## Example Interaction

```
User: I am feeling great today!
Bot: Positive

User: This is frustrating...
Bot: Negative
```
